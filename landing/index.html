On the roadmap

Already useful, with more ways to act coming. These features are planned. Each one helps your team turn a replay into a faster fix.

Planned
Replay to test case

Turn an error session into a test case with the steps, screen state, and failure details included.

Planned
Bring replays into your tools

Let your coding tools pull session context, failure details, and replay links without switching tabs.

Planned
Record first, keep what matters

Keep a short buffer of every session, then save the ones with errors, rage clicks, or support tickets.

Planned
Share replay links

Send a secure link for one session so teammates can open the replay without joining the whole workspace.

Planned
Turn on recording without a deploy

For supported sites, turn on recording from Orange Replay without editing app code or adding a tag manager.

Planned
Search sessions by problem

Ask for sessions where customers struggled with checkout, pricing, onboarding, or any flow you care about.

Planned
Heatmaps

See where people click and scroll so you can spot confusing parts of a page.

Planned
More review tools

Add dead-click markers, activity highlights, clips, and comments so teams can review a session together.
