What changed

  • The object is flat and hardware-like, so it avoids the glossy marble look.
  • The outer face is locked to a real circle, so it cannot become an oval.
  • The eye shape comes from LED bulbs, so the concept is visible at every size.
  • The card has clear actions, so it reads like a shareable live-session component.

Build notes for the real landing page

Outer mark Use aspect-ratio: 1, border-radius: 50%, and avoid transform scaling.
Pixel grid Generate dots from one helper so large, small, and tiny marks stay related.
Glow Only lit pixels glow. The housing stays dark and matte.
Motion Slow flicker and breathing only. Respect reduced motion.
